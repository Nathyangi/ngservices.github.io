Building a Simple Network Monitoring Lab

Published on October 20, 2025 • Category: Networking

This lab teaches you how to set up a small virtual network monitoring environment using open-source tools like Wireshark, Nmap, and tcpdump. You’ll gain hands-on experience capturing packets, scanning hosts, and analyzing traffic flow between virtual machines.

Steps Overview

  1. Set up two Ubuntu VMs in VirtualBox or VMware.
  2. Install Wireshark and Nmap on one VM.
  3. Generate traffic using ping, curl, and SSH between both machines.
  4. Capture and analyze traffic with Wireshark filters.

Tools Needed

  • Wireshark
  • Nmap
  • VirtualBox / VMware
  • Ubuntu 22.04 ISO
