1. What is an IP Address?

An IP (Internet Protocol) address is a unique number assigned to each device in a network. It helps route data from one computer to another across the internet or a local network. Example: 192.168.1.10

2. IPv4 vs IPv6

- IPv4: 32-bit system with 4.3 billion possible addresses (e.g., 192.168.0.1). - IPv6: 128-bit system, offering an enormous pool of unique addresses (e.g., 2001:0db8::1).

IPv6 was developed to solve the problem of IPv4 exhaustion and includes improved features like auto-configuration and security.

3. Understanding Subnetting

Subnetting is the technique of dividing a large network into smaller parts called subnets. Each subnet can function as its own network, allowing better management, security, and traffic control.

Analogy: Imagine an apartment building (the network). Each apartment has a unique room number (subnet) to help deliver mail (data) efficiently.

4. Subnet Masks

A subnet mask tells us which part of an IP address identifies the network and which part identifies the host.

Example: 192.168.1.0 /24 means the first 24 bits define the network.
